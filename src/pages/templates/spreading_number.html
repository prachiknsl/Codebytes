Spread the Digit from a Number without using any Arithmetic Operator or String

Spread out all of the digits from the given number, without using any Arithmetic Operator or String. Example: Take the number 123456 and it became 1, 2, 3, 4, 5 and 6.


In C library function int scanf (const char *format…) reads formatted input from stdin. It have some special property to take input from stdin.


Width Specification:

  • Scanf function has width specification that consists of characters between the % and type field specified. In the width specification characters should be positive integer that is called width field, and it is less then range of type field. For Example: scanf (“%5d”, &num).


Width Field:

  • The width field is a positive integer controlling the maximum number of characters to be read form the console, and rest of the characters are stored in Buffer.

  • In Example scanf ("%5d", &num), 5 is Width Field, that will take only 5 characters from the console and rest of the characters are store in buffer.

WAP of stdin/stdout:

#include"stdio.h"
int main()
{
int num, flag=1;
printf( "Enter the Number\t" );
while( scanf ("%1d", &num) != -1 )
{
if(flag) printf("\nOutput is:\t");
printf("%d, ",num);
flag=0;
}
return 0;
}

Output:

Enter the Number 123456 Output is: 1, 2, 3, 4, 5, 6,
